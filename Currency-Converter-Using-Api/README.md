# Currency Converter (Using API)

A simple and responsive Currency Converter Web App that allows users to convert one currency into another in real-time using an external API.

## Features

- Convert between multiple international currencies
- Real-time exchange rates using API
- Country flags for better UX
- Responsive and modern UI design
- Fast and lightweight (pure HTML, CSS, JavaScript)

## Technologies Used

- HTML5 – Structure of the web app
- CSS3 – Styling and responsive design
- JavaScript (Vanilla JS) – Logic and API handling
- Currency API – For live exchange rates

## Project Structure

Currency-Converter-Using-Api/
│── index.html       # Main HTML file
│── style.css        # Styling file
│── script.js        # Main logic & API handling
│── codes.js         # Currency & country codes mapping

## How It Works

User selects:
"From" currency
"To" currency
Enters the amount
Clicks Convert
App fetches real-time exchange rate from API and displays result

## API Used

The project uses a free currency API:

https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api

## UI Preview

- Dropdowns for selecting currencies
- Input field for amount
- Conversion result displayed instantly
- Country flags shown dynamically
